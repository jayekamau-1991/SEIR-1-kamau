Week 5 — Terraform IVPAD Sequence

SEIR-1 | TKO Group | April 14, 2026

IVPAD Steps

1. INIT     - terraform init     - downloads plugins
2. VALIDATE - terraform validate - checks files for errors  
3. PLAN     - terraform plan     - previews what will be built
4. APPLY    - terraform apply    - actually builds it
5. DESTROY  - terraform destroy  - tears it all down

Things to Remember

- Always run in order: init, validate, plan, apply, destroy
- Variables with no default need -var flag
- Never delete tfstate manually
- gcloud auth application-default login needed for Terraform
- Always destroy when done — stopped VMs still cost money
